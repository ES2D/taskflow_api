# TaskFlow API

API REST desarrollada con Spring Boot para la gestión de proyectos y tareas.

## Tecnologías

- Java 21
- Spring Boot 4
- Spring Data JPA
- PostgreSQL
- Maven
- Lombok

## Funcionalidades

### Projects
- Crear proyecto
- Listar proyectos
- Consultar proyecto por ID
- Actualizar proyecto
- Eliminar proyecto

## Endpoints

### Obtener todos los proyectos

GET /api/projects

### Obtener proyecto por ID

GET /api/projects/{id}

### Crear proyecto

POST /api/projects

### Actualizar proyecto

PUT /api/projects/{id}

### Eliminar proyecto

DELETE /api/projects/{id}

## Base de datos

PostgreSQL
